### Ветка `start_labs`

В этой ветке находятся **нулевые лабораторные работы**, предназначенные для:
- понимания структуры репозитория и подхода к работе
- изучения основ работы с **gmsh** (геометрия и сетки)
- изучения основ работы с **VTK** (визуализация)
- изучения основ работы с **FEniCS** (вычисления)
